把课件变成双语对照 PDF

DocTwin 是一个面向 AI Agent 的文档翻译 Skill：左侧保留原文，右侧生成译文， 优先保护公式、代码、图表、表格和页面结构。

查看项目 下载 v0.1.0

Original

int main() {
  return 0;
}

简体中文

int main() {
  return 0;
}

核心能力

双语对照左原文、右译文，适合啃课件和论文。
版式优先尽量保留公式、代码、图表和表格位置。
AI Agent 工作流Codex 一等适配，其他能运行脚本的 Agent 也可使用。
术语一致使用 translation memory 和 glossary 控制专业词。
渲染质检交付前检查页面截图，修复缺译、错位和乱码。
版权水印默认给最终 PDF 加轻量 DocTwin 署名水印。

快速安装

mkdir -p ~/.codex/skills
cp -R skills/doctwin-translate ~/.codex/skills/
python3 -m pip install PyMuPDF PyYAML pillow pdfplumber pypdf reportlab

在 Codex 中调用： 使用 $doctwin-translate 把这份 PDF 做成左右双语对照版。
