# 中文宣传文案

## 一句话

我做了一个 AI Agent Skill：**DocTwin Translate**。它可以把外语 PDF、课件、论文做成左右双语对照版本，尽量保留原排版、公式、代码、表格和图表。

## 长一点

很多留学生最痛苦的事情不是翻译单词，而是啃整套英文/德文课件：公式不能乱，代码不能乱，图表不能乱，排版也不能乱。

所以我做了 **DocTwin Translate**：一个给 AI Agent 用的文档翻译 Skill。你把 PDF 丢给 AI，它会按工作流生成双语对照版：左边原文，右边中文译文。它会使用术语表、翻译记忆、PDF 排版引擎和渲染检查，尽量保留原来的页面结构。

适合：

- 留学生啃 lecture slides
- 看英文/德文/日文技术课件
- 阅读有公式、代码、表格、图表的 PDF
- 想要双语对照，而不是只有一份机器翻译文本

项目地址：

GitHub: https://github.com/0xTaoZ/doctwin-codex-skill

生成的公开样例默认带有 DocTwin 轻水印，用来保留项目和作者署名。

## 注意

PDF 版式非常复杂，DocTwin 不是魔法，也不承诺每个文件都 100% 完美。它的定位是 layout-first：优先保留排版，再用 AI 翻译和 QA 去不断修正。
